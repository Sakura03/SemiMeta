# Semi-Supervised Learning with Meta-Gradient

By Xin-Yu Zhang, Taihong Xiao, Haolin Jia, Ming-Ming Cheng, and Ming-Hsuan Yang.

## Prerequisite

| **Package**    | **version**  |
|----------------|--------------|
